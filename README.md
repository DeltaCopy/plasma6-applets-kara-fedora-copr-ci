  # plasma6-applets-kara-fedora-copr-ci
  RPM build spec for building Kara on the Fedora COPR.

  This auto triggers Fedora Copr builds: <https://copr.fedorainfracloud.org/coprs/deltacopy/plasma6-applets-kara/builds>

  Upstream: https://github.com/dhruv8sh/kara

  ## Active releases available

  - Fedora 42
  - Fedora 43
  - Fedora rawhide

  # Instructions

  Enable the COPR repository then install the package.

  <pre>
  sudo dnf copr enable deltacopy/plasma6-applets-kara
  sudo dnf in darkly
  </pre>
